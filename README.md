# York University Steel Bridge Team Website

A modern, responsive website for the York University Steel Bridge Team built with React.

## Features

- **Landing Page**: Eye-catching hero section with team introduction
- **About Us**: Information about the team's mission and values
- **Subteams**: Detailed information about each subteam (Finance, Promotions, Drafting, Fabrication, Teams)
- **Contact/Sponsor**: Contact form and sponsorship information
- **FAQ**: Frequently asked questions with expandable answers

## Design

The website features a black and gold theme inspired by the team's logo, with:
- Metallic gold accents (#D4AF37, #FFD700)
- Black background (#000000)
- Modern, responsive design
- Smooth animations and transitions

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Technologies Used

- React 18
- React Router DOM 6
- CSS3 (Custom styling with black and gold theme)

```
src/
```
